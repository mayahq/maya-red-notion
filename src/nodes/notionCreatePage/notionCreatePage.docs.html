Description

Notion node that helps you create a new notion page. For more information visit: Create Page API docs

Inputs

url string
URL of the parent database of parent page
parent_type enum
database: if parent is a database or page: if parent is a page
properties json
The properties object of the page conforming to the properties of its parent
children json
Children items for the page
icon json
To specify the icon of the page
cover json
To specify the cover of the page

Outputs

payload various
Contains the http response from notion create page API
